Project Overview

This project analyzes traffic accident data using Python, Pandas, and Matplotlib to identify patterns in accident causes, weather conditions, and vehicle types.

Outputs:

  • Graphs showing accident frequency by cause
  • Analysis of accident trends based on weather
  • Visualization of accident-prone locations
